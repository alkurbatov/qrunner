# qrunner
Simple runner for applescripts.

# Dependencies
* Qt v5.8
* qmake
* macdeployqt (for release build).

# Build
```bash
# Generate build routine.
$ qmake

# Debug build.
$ make debug

# Release build with a .dmg image.
$ make release
```
